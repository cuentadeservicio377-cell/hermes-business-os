# Guía de Usuario — Hermes Business OS

## Primeros Pasos

### 1. Iniciar conversación

Abre Telegram y escribe a tu bot:
```
/start
```

Hermes te saludará y te guiará por el onboarding.

### 2. Configurar tu empresa (si no lo hiciste)

```
Hola, soy nuevo. Configura mi empresa.
```

Hermes te hará preguntas sobre tu negocio y generará la configuración automáticamente.

### 3. Conectar Google Workspace

```
Conecta Google Workspace
```

Hermes creará automáticamente:
- Spreadsheet "Índice de Proyectos"
- Carpeta "Hermes OS" en Drive
- Estructura de carpetas por cliente

## Uso Diario

### Ventas

**Registrar cliente:**
```
Registré un cliente nuevo: Juan Pérez, quiere una boda para 100 personas el 15 de octubre
```

**Cotizar:**
```
Cotiza una boda para 100 personas, paquete intermedio
```

**Seguimiento:**
```
¿Qué pasó con Juan Pérez?
```

**Ver pipeline:**
```
Muestra mi pipeline
```

### Operaciones

**Crear proyecto:**
```
Crea proyecto "Boda Juan y María" para el 15 de octubre
```

**Ver tareas:**
```
¿Qué tareas tenemos pendientes?
```

**Completar tarea:**
```
Marca la tarea "Confirmar lugar" como completada
```

### Documentos

**Generar documento:**
```
Genera contrato de servicios para Juan Pérez
```

**Ver templates:**
```
¿Qué documentos puedes generar?
```

### Finanzas

**Reporte:**
```
¿Cuánto facturamos este mes?
```

**Presupuesto:**
```
Presupuesto del proyecto PROJ-001
```

## Comandos Rápidos

| Comando | Descripción |
|---------|-------------|
| `/empresa` | Ver configuración actual |
| `/departamentos` | Listar departamentos activos |
| `/clientes` | Listar clientes |
| `/proyectos` | Listar proyectos |
| `/estado` | Resumen del negocio hoy |
| `/reporte ventas` | Métricas de ventas |
| `/reporte ops` | Estado de operaciones |
| `/documentos` | Templates disponibles |

---

## Tips

1. **Habla natural** — No necesitas comandos exactos. Hermes entiende intenciones.
2. **Contexto persistente** — Si mencionas "Juan Pérez", Hermes lo recuerda en la conversación.
3. **Voz** — Puedes enviar audios por Telegram. Hermes los transcribe.
4. **Documentos** — Siempre pide aprobación antes de enviar cotizaciones.
5. **Alertas** — Hermes te avisará automáticamente de deadlines y seguimientos.
